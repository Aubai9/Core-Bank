# 🏦 Console Bank System (C++)

## 📌 About The Project
This project is a robust, terminal-based **Banking Management System** built from scratch using C++. 
It serves as a comprehensive demonstration of **CRUD operations** (Create, Read, Update, Delete) applied to financial records. The system manages client data efficiently in-memory using dynamic Data Structures (`std::vector`) and provides a professional, user-friendly Console UI.

## 🚀 Key Features
- **Client Management:** Add new clients with unique Account Numbers, update existing info, or delete accounts securely.
- **Financial Transactions:** Seamlessly perform **Deposit** and **Withdrawal** operations with real-time balance validation (preventing overdrafts).
- **Advanced UI/UX:** Features a hierarchical Menu System (Main Menu & Transaction Menu) with screen cleaning and organized table formatting using `iomanip`.
- **Search & Reporting:** Ability to find specific client records and view a comprehensive list of all clients with a total balance summary.
- **Data Integrity:** Includes robust validation to prevent duplicate account numbers and invalid transaction amounts.

## 🛠️ Built With
* **Language:** C++
* **Libraries:** `<iostream>`, `<vector>`, `<string>`, `<iomanip>`, `<cstdlib>`

## 🧠 Technical Highlights
- **Modular Design:** Each system operation is isolated into dedicated functions for clean and maintainable code.
- **Data Structures:** Leveraged `struct` for data modeling and `std::vector` for dynamic data management.
- **Logic Implementation:** Developed custom algorithms for searching, filtering, and updating records without external database dependencies.
